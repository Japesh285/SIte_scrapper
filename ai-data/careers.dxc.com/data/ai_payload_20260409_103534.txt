Project description:
 
We are looking for a talented .NET & React.js Developer to build, enhance, and maintain scalable web applications. You need to have strong backend development experience using .NET technologies along with hands-on expertise in React.js to deliver high-quality, responsive, and user-centric applications.
 
Responsibilities:
 
Design, develop, and maintain web applications using C#, ASP.NET, and ASP.NET Core 
Develop dynamic, responsive, and reusable UI components using React.js 
Integrate React front-end applications with RESTful APIs and backend services 
Collaborate with UI/UX designers to implement modern user interfaces 
Optimize application performance, scalability, and security 
Write clean, maintainable, and well-documented code 
Perform code reviews, debugging, and unit testing 
Participate in Agile/Scrum development processes 
Troubleshoot and resolve application issues
 
Mandatory skills:
 
Strong experience in C#, .NET, ASP.NET / ASP.NET Core – at least 7 years of experience 
Proficiency in React.js, JavaScript (ES6+), HTML5, and CSS3 
Experience with REST APIs and JSON 
Solid understanding of state management (Redux, Context API, etc.) 
Experience with SQL Server or other relational databases 
Familiarity with Git and version control workflows 
Understanding of responsive design and cross-browser compatibility
