Job Description:
 
Role Overview
 
We are looking for an SAP ABAP Consultant with strong experience across SAP ERP (ECC or S/4HANA) and SAP Business Technology Platform (BTP). The candidate will be responsible for designing, developing, and optimizing custom solutions, integrations, and extensions that span both on-premise and cloud landscapes.
 
 
Key Responsibilities
 
1. 
ABAP Development (ECC & S/4HANA)
 
 
Develop and enhance ABAP objects: Reports, Interfaces, Conversions, Enhancements, Forms (RICEFW).
 
Expertise in OO-ABAP, AMDP, CDS Views, OData services, and BOPF.
 
Performance optimization for ABAP code, SQL traces, dumps, and runtime analysis.
 
Work on Proxy, IDoc, and integration objects with middleware.
 
Work on SmartForms, Adobe Forms, Workflow enhancements.
 
 
2
. SAP BTP Development
 
 
Build and manage cloud extensions using ABAP RESTful (RAP), Node.js/java etc
 
Develop OData V4 services, CDS-based data models, and API consumption.
 
Integrate BTP applications with SAP S/4HANA via BTP Destination Service, Connectivity Service, Event Mesh, API Management, or Integration Suite.
 
 
3. 
Integration & Architecture
 
 
Design extension architecture for hybrid systems (on-prem ERP + BTP).
 
Work with SAP API Business Hub, S/4HANA APIs, and event-driven patterns.
 
Support SAP Cloud ALM or Solution Manager for transport and monitoring.
 
 
4
. DevOps, Quality, and Lifecycle
 
 
Use gCTS, CI/CD pipelines in BTP or Git-based repositories.
 
Follow coding standards, documentation, and secure development practices.
 
Participate in code reviews, testing cycles, and production deployments.
 
 
 
Required Skills
 
 
Strong knowledge of ABAP, OO-ABAP, CDS, OData, RAP/RESTful ABAP.
 
Hands-on with SAP BTP services, CAP/RAP, and cloud integration concepts.
 
Experience with Fiori/UI5 development and extending standard apps.
 
Practical experience in S/4HANA data models, CDS annotations, and HANA optimization.
 
Good understanding of security, destinations, OAuth, JWT, and principal propagation.
 
 
 
Preferred Skills
 
 
Knowledge of SAP Event Mesh, BTP Workflow, Automation Pilot, or Build Apps.
 
Certification in ABAP for SAP HANA, BTP Developer, or Extensibility.
 
Experience in agile delivery, Jira, GitHub/GitLab, and transport management.
 
Working Knowledge with SAP Integration Suite (CPI). CAP (Cloud Application Programming Model)
 
 
 
Experience Level
 
 
With skills of Deep ABAP + solutioning skills –
10+ years:
 
 
 
BTP Hands on  & architecture exposure –
3-4 + years
