Job Description: 
        


Job Summary
 
Highly skilled Lead UI Developer with a primary focus on ReactJS and associated technologies to join our dynamic team. The ideal candidate will have extensive experience in developing user interfaces, leading development teams, and collaborating with cross-functional teams to deliver high-quality software solutions. This role requires a deep understanding of front-end technologies, UI/UX principles, and software development best practices.
Key Responsibilities
 
Strategy
•    Lead the design and development of user interfaces using ReactJS, Redux, Redux-Saga, and Redux-Thunk.
•    Implement and maintain RESTful services and ensure seamless integration with front-end components.
•    Develop and maintain CSS styles and ensure responsive design across different devices.
•    Utilize Node.js and npm package managers 
•    Write and maintain advanced JavaScript code, ensuring high performance and compatibility across browsers.
•    Conduct unit testing using Jest with Enzyme and React Testing Library to ensure code quality and reliability.
•    Configure and manage HTTP servers and Nginx for optimal performance and security.
•    Implement microfrontend architecture to enable scalable and maintainable codebases.
•    Understands and implements Groups Technology standards.


Business
•    Collaborate with UX/UI designers to translate user flows, wireframes, prototypes, and mock-ups into functional user interfaces.
•    Maintain and update style guides to ensure consistent design language across the application.
•    Stay updated with the latest industry trends and technologies to ensure the application remains modern and competitive.
•    Align technical projects with business goals and objectives to deliver value to stakeholders.
•    Work with product owners and leads to prioritize features and improvements based on business needs and user feedback.


Processes
eSDLC Process, Scrum/Agile, Versioning and Release Management
 
People & Talent
•    Provide mentorship and guidance to other UI developers and foster a collaborative team environment.
•    Participate in code reviews and provide constructive feedback to team members.
•    Foster a culture of continuous learning and improvement within the development team.
•    Assist in the recruitment and onboarding of new team members.
•    Conduct regular performance reviews and provide feedback to team members of different roles.


Risk Management
•    Perform code quality checks using SonarQube or similar tools to ensure maintainability and adherence to coding standards.
•    Utilize basic Linux/Unix commands for server management and deployment tasks.
•    Assist in troubleshooting and resolving performance issues and bugs.
•    Ensure compliance with security best practices and data protection regulations.
•    Implement measures to mitigate risks associated with software development and deployment.
•    Identifies and reports potential issues and risks in delivery, and works with team to provide mitigation steps or fix options to close the risk.


Governance
•    Utilize GIT for version control and collaborate with the team using branching and merging strategies.
•    Integrate continuous integration and continuous deployment (CI/CD) pipelines using Jenkins.
•    Ensure adherence to development processes and best practices.
•    Document technical specifications and development processes.
•    Ensure that all development activities comply with organizational policies and regulatory requirements.
 
Regulatory & Business Conduct


•    Display exemplary conduct and live by the Group’s Values and Code of Conduct. 
•    Take personal responsibility for embedding the highest standards of ethics, including regulatory and business conduct, across Standard Chartered Bank. This includes understanding and ensuring compliance with, in letter and spirit, all applicable laws, regulations, guidelines and the Group Code of Conduct.
•    Effectively and collaboratively identify, escalate, mitigate and resolve risk, conduct and compliance matters.


Key stakeholders


•    Delivery Lead, Scrum Lead, Scrum Team, Product Owner, Production Engineering Team


Other Responsibilities
Embed Here for good and Group’s brand and values in Do the right thing, Never Settle, Better together [India / IT-Projects-IC-T&A / ESG]; Perform other responsibilities assigned under Group, Country, Business or Functional policies and procedures; Multiple functions (double hats)
 
Skills and Experience
 
•    Preferably 7-10 Years of experience in Application Development using ReactJS with Redux, CSS, REST(see technical competence below)
•    Strong understanding of fundamental architecture and design principles, object-orientation principles, and coding standards. 
•    Experience in building extensible and scalable solutions
•    Strong analytical and problem-solving skills
•    Strong verbal and written communication skills
•    Good knowledge in DevOps, CI-CD
•    Understanding on source control, versioning, branching etc.
•    Experienced in Agile methodology and Waterfall models
•    Strong experience in Application Delivery, that also includes Production Support
•    Good presentation and documentation skills
•    Ability to learn and adapt to new technologies and frameworks
•    Awareness about Release Management
•    Strong team player who can collaborate effectively with relevant stakeholders
Technical Competence
Must Have
•    ReactJS with Redux
•    Redux middleware like Redux-Saga, Redux-Thunk, etc.
•    Knowledge on WebServices, REST
•    CSS processors like Sass, Less, etc.
•    Nodejs and package managers (npm/yarn)
•    JavaScript Utility Libraries like Underscore, Lodash, etc.
•    Unit tests using Jest with Enzyme or React Testing Library
•    Understanding of GIT, Jenkins, SonarQube or other related tools
•    Strong in Advanced JavaScript (ES6), HTML5, CSS3
•    Good knowledge of web standards, security, accessibility, and browser compatibility
•    Understanding of continuous build concept with GIT, Jenkins, SonarQube or other related tools
•    Knowledge of Linux commands
•    Knowledge in configuring and maintaining HTTP servers (Apache HTTP/NGINX)
•    Quickly learn and adapt to new tools, technologies, and frameworks related to UI/UX
 
Good to Have
•    Basic Linux/Unix Commands
•    GitHub Co-pilot
•    Designing UI elements like input controls, navigable components etc.
•    Knowledge in creating user flows, wireframes, prototypes, and mock-ups
•    Experience in collaborating with product, engineering, and management functions
•    Developing and maintaining style guide documents
•    Presenting ideas, concepts, and solutions to relevant stakeholders
•    Knowledge on Microfrontend and with an implementation knowledge is desirable
Qualifications
 
•    EDUCATION: ANY DEGREE
•    TRAINING: ANY RELEVANT TRANING BASED ON THE SKILL REQUIREMENTS ABOVE    
•    CERTIFICATIONS: ANY RELEVANT CERTIFICATIONS BASED ON THE SKILL REQUIREMENTS ABOVE
•    LANGUAGES: ReactJS with Redux, Redux-Saga, Redux-Thunk, React Testing Library, etc, REST Services, CSS, Node and npm package managers, Advanced JavaScript, Unit Tests: Jest with Enzyme, React Testing Library, HTTP and Nginx config 
Role Specific Technical Competencies
 
•    React JS with Redux
•    REST Services & CSS
•    Microservices
•    Containerization / Kubernetes / Docker (OCP/SKE)
•    HTTP commands
•    Other related java frameworks/libraries
